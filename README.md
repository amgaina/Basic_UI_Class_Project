# ULM Neighborhood Safety Watch

A community safety awareness website for the class project for Internet Programming (CSCI 3030).

## Features

- **Safety Tips**: Practical advice for home, walking, and vehicle safety
- **Alerts**: Report and view neighborhood safety concerns
- **Resources**: Emergency contacts and local safety information

## Technologies

- React.js
- HTML5
- CSS3
- JavaScript (ES6+)

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the app: `npm start`

## Color Scheme

- Primary: Crimson (#990033)
- Secondary: White, Light Gray
